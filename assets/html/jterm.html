JTERM

Cross Platform

Because JTerm is written in Java, any computer with Java installed can instantly run the JTerm application. The build process uses Apache's Maven, which also runs off of Java. So that means you can also help develop this project on any OS!

Simply ROOT

JTerm has many of the same features as the ROOT project, so you're not missing out on anything. JTerm comes with a lot of extra utilities, such as a work-in-progress socket server + client, and also some OS-dependent features like a process viewing command.

Contributed

JTerm is an open-collaberated project, ensuring that multiple people's input are recieved on updates. There are more and more features that we need help on, so come help out!
